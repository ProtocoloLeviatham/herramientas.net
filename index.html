Código de Ética y Descargo de Responsabilidad

Este repositorio ("Leviatham") está destinado **exclusivamente para fines educativos, de investigación y de auditoría de seguridad autorizada** (Hacking Ético).

Al aceptar y continuar, usted declara que entiende y se compromete a lo siguiente:

  • Uso Legal: Se compromete a no utilizar ninguna herramienta, técnica o información de este sitio para actividades ilegales o maliciosas.
  • Prohibido el Ataque: No utilizará este conocimiento para atacar, penetrar o dañar sistemas, redes o aplicaciones para los que no tenga permiso explícito por escrito.
  • No Dañar: No causará interrupciones de servicio (DoS), pérdida de datos o cualquier tipo de daño a terceros.
  • Responsabilidad Total: Usted es el único responsable de sus acciones. El creador (@PapaLucker111) y los contribuidores de este sitio no se hacen responsables de ningún mal uso o daño causado por la información aquí presentada.
  • Propósito Educativo: El único fin es aprender a defender sistemas, no a destruirlos.
  • Consentimiento: El "Hacking Ético" requiere consentimiento. Siempre obtenga autorización antes de probar la seguridad de cualquier sistema.

El incumplimiento de estas reglas puede tener graves consecuencias legales. Piense antes de actuar.

Recolección de Información (OSINT)

Maltego

Herramienta líder en minería de datos y OSINT. Permite visualizar gráficamente las relaciones entre personas, dominios, IPs y más.

Casos de Uso:

Ciberinteligencia, análisis forense, mapeo de infraestructura de un atacante.

Intermedio Comercial (Ver. Community)
Visitar Web

TheHarvester

Script para recolectar emails, subdominios, nombres de empleados y hosts desde fuentes públicas (Google, Bing, Shodan, etc.).

Casos de Uso:

Fase inicial de pentesting, recolección de correos para phishing (ético), descubrimiento de activos.

Comando Esencial 
theharvester -d [dominio.com] -l 500 -b [google,bing]
Principiante Open Source
Ver en GitHub

Shodan

El "Google de los hackers". Un motor de búsqueda que permite encontrar dispositivos conectados a Internet (cámaras, servidores, IoT).

Casos de Uso:

Encontrar dispositivos IoT vulnerables, monitorizar activos expuestos de una empresa.

Ejemplo de Búsqueda 
# Busca cámaras web en un país:
webcam country:"ES"

# Busca servidores Apache vulnerables:
apache version:"2.4.1"
Principiante Freemium
Visitar Web

Análisis de Redes

Wireshark

El analizador de protocolos de red más popular. Permite capturar y examinar tráfico de red en tiempo real a nivel de paquete.

Casos de Uso:

Análisis de tráfico, debugging de red, detección de intrusiones, análisis de malware.

Filtro Esencial 
# Filtrar por dirección IP (origen o destino):
ip.addr == [192.168.1.1]

# Filtrar por protocolo específico:
tcp.port == 80
Intermedio Open Source
Visitar Web

Nmap

(Network Mapper) La herramienta esencial para el descubrimiento de redes y auditoría de seguridad. Escanea puertos, detecta servicios y S.O.

Casos de Uso:

Escaneo de puertos, descubrimiento de hosts, huella digital de S.O., inventario de red.

Comandos Esenciales 
# Escaneo rápido y sigiloso (SYN):
nmap -sS [IP_Objetivo]

# Detección de servicios y versiones:
nmap -sV -sC [IP_Objetivo]
Principiante Open Source
Visitar Web

Metasploit

El framework de explotación más conocido. Se usa para desarrollar, probar y ejecutar exploits con fines de pentesting y validación de seguridad.

Casos de Uso:

Explotación de vulnerabilidades, desarrollo de exploits, post-explotación, validación de parches.

Estructura de Uso 
msfconsole
> use [exploit/ruta/del/exploit]
> set RHOSTS [IP_Objetivo]
> set PAYLOAD [payload/ruta/payload]
> exploit
Avanzado Open Source (Framework)
Visitar Web

Análisis de Vulnerabilidades Web

OWASP ZAP

(Zed Attack Proxy) Escáner de seguridad web open-source mantenido por OWASP. Ideal para encontrar vulnerabilidades en apps web.

Casos de Uso:

Escaneo DAST, "spidering" de sitios, intercepción de peticiones, fuzzing de parámetros.

Modo de Uso 
# 1. Configurar el proxy
(Navegador -> localhost:8080)
# 2. Exploración (Spidering)
(Usar la app mientras ZAP captura)
# 3. Escaneo Activo
(Click derecho en el sitio -> "Active Scan")
Intermedio Open Source
Visitar Web

Burp Suite

La plataforma integrada "todo en uno" para pruebas de seguridad en apps web. Su proxy es el estándar de la industria.

Casos de Uso:

Intercepción y modificación de tráfico web, pentesting manual, escaneo de vulnerabilidades.

Modo de Uso 
# 1. Configurar el proxy
(Navegador -> localhost:8080)
# 2. Módulo Proxy -> Intercept
(Activar "Intercept is on")
# 3. Enviar a Módulos
(Click derecho -> Send to Repeater/Intruder)
Intermedio Comercial (Ver. Community)
