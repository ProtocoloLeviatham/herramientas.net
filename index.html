Repositorio de Hacking Ético

Recolección de Información (OSINT)

Maltego

Herramienta líder en minería de datos y OSINT. Permite visualizar gráficamente las relaciones entre personas, dominios, IPs y más.

Casos de Uso:

Ciberinteligencia, análisis forense, mapeo de infraestructura de un atacante.

Intermedio Comercial (Ver. Community)
Visitar Web

TheHarvester

Script para recolectar emails, subdominios, nombres de empleados y hosts desde fuentes públicas (Google, Bing, Shodan, etc.).

Casos de Uso:

Fase inicial de pentesting, recolección de correos para phishing (ético), descubrimiento de activos.

Principiante Open Source
Ver en GitHub

Shodan

El "Google de los hackers". Un motor de búsqueda que permite encontrar dispositivos conectados a Internet (cámaras, servidores, IoT).

Casos de Uso:

Encontrar dispositivos IoT vulnerables, monitorizar activos expuestos de una empresa.

Principiante Freemium
Visitar Web

Análisis de Redes

Wireshark

El analizador de protocolos de red más popular. Permite capturar y examinar tráfico de red en tiempo real a nivel de paquete.

Casos de Uso:

Análisis de tráfico, debugging de red, detección de intrusiones, análisis de malware.

Intermedio Open Source
Visitar Web

Nmap

(Network Mapper) La herramienta esencial para el descubrimiento de redes y auditoría de seguridad. Escanea puertos, detecta servicios y S.O.

Casos de Uso:

Escaneo de puertos, descubrimiento de hosts, huella digital de S.O., inventario de red.

Principiante Open Source
Visitar Web

Metasploit

El framework de explotación más conocido. Se usa para desarrollar, probar y ejecutar exploits con fines de pentesting y validación de seguridad.

Casos de Uso:

Explotación de vulnerabilidades, desarrollo de exploits, post-explotación, validación de parches.

Avanzado Open Source (Framework)
Visitar Web

Análisis de Vulnerabilidades Web

OWASP ZAP

(Zed Attack Proxy) Escáner de seguridad web open-source mantenido por OWASP. Ideal para encontrar vulnerabilidades en apps web.

Casos de Uso:

Escaneo DAST, "spidering" de sitios, intercepción de peticiones, fuzzing de parámetros.

Intermedio Open Source
Visitar Web

Burp Suite

La plataforma integrada "todo en uno" para pruebas de seguridad en apps web. Su proxy es el estándar de la industria.

Casos de Uso:

Intercepción y modificación de tráfico web, pentesting manual, escaneo de vulnerabilidades.

Intermedio Comercial (Ver. Community)
Visitar Web

Sistemas Operativos

Kali Linux

La distribución de Linux de referencia para Pruebas de Penetración y Auditoría de Seguridad. Viene pre-cargada con cientos de herramientas.

Casos de Uso:

Plataforma completa de pentesting, auditoría de seguridad, laboratorios de hacking.

Principiante Open Source
Visitar Web

Parrot OS

Una distribución de Linux robusta enfocada en seguridad, privacidad y desarrollo. Es una gran alternativa a Kali, muy ligera y completa.

Casos de Uso:

Alternativa a Kali, enfocada en privacidad, análisis forense, desarrollo seguro.

Principiante Open Source
